# Sobre el proyecto

Corggio es un template de página web front-end ecommmerce para una empresa que comercializa y distribuye artículos de ferretería. El stack tecnológico con el cual está contruido incluye HTML, CSS, SASS e incorpora el Bootstrap como framework principal.
